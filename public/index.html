A meet up group for coders

Learning how to code can be a very daunting experience. Here at Wigan Coders, we try to help you understand any technical jargon, and help get you started on the correct path into learning how to code. Our Meet-Up group is led by two full time web developers with industry experience, ensuring you are taught the best in up to date coding practices.

The most up-to-date in industry standards

HTML 5
CSS 3
JavaScript

If it's websites or applications you would like to build, or gain knowledge in how this is done, then this is the place for you. We have commercial knowledge is HTML, CSS and JavaScript. These three coding practices combined can help us build some beautiful, responsive websites and applications.

A close community

Together

Here at Wigan Coders we believe that doing things together help's us all achieve more. Coding in a mob group session can really help your confidence and aspire others to speak up and ask questions where they otherwise may not.

Support

We have a huge support base here at Wigan Coders. We use SLACK, software for developers to share idea's and motivate others through chats and other technologies which help keep us together even when we are not in the same room.

Achieve

Software development can bring a huge sense of achievement. Writing your first program or website can sound very confusing, however it is very satisfying and can be very rewarding. Web development is also very flexible.

Resources to get you started straight away!

We have plenty of suggestions for online resources to get you learning how to code today!

Free online courses

Udemy

Udemy has a fantastic free section to their website. You can start a free online course in practically any coding language. If you can afford it, they have some pretty decent premium content too.

Free-Code-Camp

Free-Code-Camp is exactly that. They have a huge curriculum from complete novive to advanced developer. They have some great projects to build and also supply you with a certificate of completion.

Code Academy

Code Academy is a fantastic online resourse which has paid and free course for you to get started with. If you are completely new to coding or want to further your coding skills, this is definitely a place to come.
